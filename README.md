# Smart Notes Manager

A simple open-source project for managing academic notes and resources.

## Features

- Organize notes by subject
- Search notes
- Export notes
- Simple command-line interface

## Installation

1. Clone the repository
2. Open project folder
3. Run application

## Project Structure

- src/ : Source code
- docs/ : Documentation
- assets/ : Static resources
- examples/ : Example files
